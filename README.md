Build an interactive analytics dashboard for Renata PLC that includes:

2. **Task 2 (Full‑Stack Dashboard)**

   - Sidebar + dashboard layout with multiple visuals and filters  
   - Role‑based login (Admin, Sales Rep, Viewer)
